Definiciones

Definición: El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos, como párrafos, listas, imágenes y tablas de datos.

Definición: CSS (Cascading Style Sheets) es un lenguaje de estilo usado para describir la presentación de documentos HTML o XML, controlando el diseño visual de la web.

Definición: JavaScript es un lenguaje de programación que se usa para crear interactividad en las páginas web, desde animaciones hasta manejo de eventos del usuario.
